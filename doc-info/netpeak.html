C# NetPeak - Network Monitor (mountain peak) - v2.1 (Decc 2009)

Download:
  • Download source code: NetPeak-v2.1-src.zip
  • Download executable code: NetPeak-v2.1.exe
  • Download graphics library code: ZedGraph.dll

    • Description

    C# Application which monitors network traffic volume by IP and Port. Able to graph and export traffic information. The built-in graph requires access to the Zed graphics package via the zedgraph.dll provided above. Further information on zed can be found on their web page     http://zedgraph.org/ .

    Main NetPeak features:

  • Monitor network interface (display total traffic by IP or by Port)
  • Gaph Read or Write totals using rolling averages
  • Capture package information (packet type and size)
  • Display network statistics
  • Reports can be exported as CSV or printed.
  • Split panels
  • Panels can be resize and selective closed.
